📌 Estructura básica de un documento HTML 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta http-equiv="X-UA-Compatible" content="IE=edge" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Título de la página</title>
</head>
<body>
<!-- Contenido visible va aquí -->
</body>
</html>

🌐 Meta Tags Esenciales & Enlaces

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Una descripción concisa.">
<meta name="keywords" content="html, css, js">
<title>Mi Página</title>

<link rel="stylesheet" href="ruta/a/estilos.css">
<link rel="icon" href="favicon.ico" type="image/x-icon">

Las meta etiquetas proporcionan información crucial sobre tu página, como la codificación de caracteres, la descripción y el viewport. También se incluyen enlaces a CSS.

🧱 Contenedores y organización

<header>Encabezado</header> <!-- Semántica -->
<main>Contenido principal</main> <!-- Semántica -->
<footer>Pie de página </footer> <!-- Semántica -->
<nav>Menú de navegación</nav> <!-- Semántica -->
<div>Bloque genérico </div> <!-- No semántica -->
<section>Sección </section> <!-- Semántica -->
<article>Artículo independiente</article> <!-- Semántica -->
<aside>Contenido lateral</aside> <!-- Semántica -->

Usa estas etiquetas para organizar tu contenido de forma semántica y mejorar la estructura de tu documento.

📝 Encabezados y Párrafos

<h1>Título Principal</h1>
<h2>Subtítulo Nivel 2</h2>
<h3>Subtítulo Nivel 3</h3>
<h4>Subtítulo Nivel 4</h4>
<h5>Subtítulo Nivel 5</h5>
<h6>Subtítulo Nivel 6</h6>

<p>Este es un párrafo de texto.
<br>Aquí hay un salto de línea.</p>

Define la jerarquía de tus títulos (h1 es el más importante) y estructura el texto en párrafos.

Más información: Texto

✨ Formato y Énfasis de Texto

<p><strong>Texto semanticamente importante</strong> <b>negrita</b>.</p>
<p><em>Énfasis</em> <i>cursiva</i>.</p>
<p><mark>Texto resaltado</mark>. <small>Texto pequeño</small>.</p>
<p>Código: <code>console.log("Hola");</code></p>
<pre><code>Bloque de código
con varias líneas.</code></pre>
<p>Fórmula H<sub>2</sub>O, potencia 2<sup>3</sup>.</p>

Aplica estilos de texto como negrita, cursiva, resaltado y muestra bloques de código con estas etiquetas.

Más información: Formato
